
# AI-Powered Resume Analyzer with Job Match Suggestions

## 🎯 Project Goal:
To develop an AI-based system that reads resumes, extracts key details like skills and experience using NLP, and suggests suitable job roles. The system will also provide skill gap feedback and training suggestions.

## 🔄 System Workflow:

    [1] Upload Resume (PDF or Text)
             |
             v
    [2] Extract Resume Text using PyMuPDF
             |
             v
    [3] Apply NLP to extract Skills, Education, Experience
             |
             v
    [4] Match resume info with job descriptions (semantic + keyword)
             |
             v
    [5] Show Job Suggestions + Skill Gaps + Course Links
